N sweep settings
For each run, one max-N signal sequence is generated and then evaluated across all τ. Results for smaller N are derived by truncating the last agents. A separate summary plot shows success probability vs N for τ=0 and τ=1.

Q sweep settings
For each run, one true state and one set of per-agent uniform draws are generated. For each q, signals are derived by thresholding those uniforms at q (so signals get strictly more accurate as q increases).
